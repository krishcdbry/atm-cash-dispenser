# ATM Cash Dispenser
ATM Cash Dispenser is a simple fun task.

# Content
Enter required amount and it will calculate the denominations in (INR) 2000, 500, 200, 100, 50, 20, 10, 5, 2, 1 Rs respectively

## Technlogies Used
HTML, CSS, Vanilla JavaScript (ES6)
